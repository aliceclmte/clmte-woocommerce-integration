CLMTE WooCommerce integration easily allows your customers to carbon offset their purchases in your WooCommerce shop. Read more at clmte.com.

== Description ==

CLMTE WooCommerce adds a checkbox to the WooCommerce checkout page which allows your customers to carbon offset their purchases. By linking your organisation with our plugin, CLMTE provides a solution for carbon offsetting your buissness.

== Required Plugins ==

This plugin requires WooCommerce to be installed and active to work.

== Installation ==

1. Download the clmte plugin
1. Upload the `clmte`-folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Fill in your organisation details under WooCommerce->Settings->CLMTE

== Frequently Asked Questions ==

= How do I get my API Key and Organisation ID? =

First, go to [CLMTE.com](https://clmte.com/ "CLMTE.com website") and create or log in to your organisation account. Under "Api Details" you will be able to copy your API Key and Organisation ID

= Where do I fill in my Organisation Details? =

Under WooCommerce->Settings, there is a tab called CLMTE. Click on it and you should see two fields named "Api Key" and "Organisation ID".
